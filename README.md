## 构建与运行

<details>
  <summary> dev </summary>

  安装/更新依赖：
  ```shell
  # 开发依赖
  pnpm i -D
  # 依赖  
  pnpm i --save
  ```

  运行：
  ```shell
  pnpm run dev
  ```
</details>
